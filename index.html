My app was created for drawing animated sprite. And in my app you can use different tools for drawing and transform. Drawing tools: pen, stroke line, rectangle, circle, paint bucket, mirror pen, color picker, dithering tool, lighten tool, tool for paint all pixels of same color, move tool and shape selection. Transform tool: rotate and verticaly flip. Small animation preview window in the top right corner is usefull thing for drawing animated picture. Also the main part of app is export. My app give you posibility to export your works as png, apng and gif. If you will want to know more about creating that programm you can visit my repo and read file which called README.MD
